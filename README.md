# Games 🎮

A collection of kid-friendly browser games. Plain HTML/CSS/JS, no build step — each game lives in its own folder and runs by opening its `index.html`.

`index.html` at the root is the arcade landing page linking to every game.

## Games

| Game | Description |
|------|-------------|
| [Fashion for Life](fashion-for-life/) 💅 | Timed salon styling competition — nails, makeup, hair, outfits, judged in rounds |
